# WanderAware Backend

Amplify Gen 2 backend for WanderAware.

Includes Cognito auth, AppSync/Data models, DynamoDB persistence, SES invitation/contact email, organization/team flows, admin tools, and Stripe-backed plan/device-capacity billing.

Primary product models include organizations, memberships, buildings, RFID door readers, RFID tags, participants, and alerts. Device-reader ingestion is not implemented in this migration.
